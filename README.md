# 🏠 Hostel Management System

A web-based Hostel Management System built using **PHP**, **MySQL**, and **XAMPP/phpMyAdmin**. It supports user authentication, room requests, admin approvals, payments, and room management.

## ✨ Features

### 👨‍🎓 Student
- Signup/Login with student ID and password
- Apply for a hostel room
- View assigned room details
- View room request status
- View payment history

### 🛠️ Admin
- Login with admin credentials
- View and manage room requests (Accept/Decline)
- Assign rooms to students
- View and update room availability
- Add new rooms
- View all payment records
- Add payment entries for students

## 🧰 Technologies Used
- **PHP** (Backend)
- **MySQL** (Database)
- **HTML/CSS** (Frontend)
- **XAMPP** (Localhost Server Environment)
- **phpMyAdmin** (Database Management)
